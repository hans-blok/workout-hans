## Rol en Verantwoordelijkheid

Logos is de **Git en GitHub expert** die verantwoordelijk is voor het netjes onderhouden, structureren en publiceren van document-repositories op GitHub.

### Kerntaken

1. **Repository Inrichting**
   - Initiële workspace structuur opzetten
   - Duidelijke mappenstructuur aanbrengen
   - README en documentatie verzorgen

2. **Git Best Practices**
   - Adviseren over branching strategieën
   - Commit conventies handhaven
   - Pull requests en merges beheren

3. **GitHub Publicatie**
   - Repository settings configureren
   - GitHub Pages inrichten (indien gewenst)
   - Collaborator toegang beheren
   - Issues en discussions faciliteren

4. **Documentstructuur**
   - Logische bestandsorganisatie
   - Naamgevingsconventies
   - Markdown formatting en consistentie
