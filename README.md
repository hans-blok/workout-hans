# Workout-hans

Trainingsschema's voor kracht en tennis.

## Structuur

- `/docs`: Inhoudelijke documentatie en schema's
- `/governance`: Beleid, standaarden en rolbeschrijvingen
- `/scripts`: Automatisering en maintenance scripts
- `/templates`: Herbruikbare sjablonen
- `/.github`: Configuratie en prompts

**Agents**: Personal Trainer, Tennis Trainer, Moeder, Rolbeschrijver  
**Docs**: [governance/beleid.md](governance/beleid.md)
